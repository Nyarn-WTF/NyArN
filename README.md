Nya-Alert-Neko is a build alert watchdog.

# Instalation
1. Install go env -> [https://golang.org/](https://golang.org/)
2. Clone this repo \
   `git clone this repo`
3. build & install
   ```
   cd NyArn
   go build
   go install
   ```
4. Add install-path \
   In ~/.zshrc \
   `
   export PATH="$PATH:/home/[username]/go/bin"
   ` \
   then, \
   `source ~/.zshrc`
